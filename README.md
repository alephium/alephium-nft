# Alephium NFT

## Install

```
npm install
```

## Getting Started

First, run the development server:

```
npm run dev
# or
yarn dev
```

## Start a devnet

```
npm run start-devnet // this will start a devnet for smart contract tests
```

## Stop/restart devnet

```
npm run stop-devnet
npm run restart-devnet
```

## Testing Smart Contract

```
npm test:contracts
```

or

```
npm test:contracts -- nft-marketplace.test.ts
```
